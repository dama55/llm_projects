
文章が途中で途切れてしまう問題を解決する

`index.html`で`max_tokens`を512で指定していたっぽい．
値を増やすか，項目自体を消すのもあり．

```
// OpenAI互換のリクエストJSON（gateway側でmodel補完やsystem挿入をするので最小でもOK）
const body = {
	stream: true, // ストリームで返してもらう
	messages: [
		{ role: "user", content: prompt }
	],
	temperature: 0.7,
	top_p: 0.9,
	max_tokens: 1500
};
```


途中で途切れる問題はある程度解決されていそう

```
Prompt: なるべく長く物語を生成してください．

はい、では少し長い物語を生成してみましょう。 物語のタイトル：「星に願いを」 遠い昔、小さな村が一つありました。その村は山に囲まれた美しい場所にあり、村の人々は日々の生活を楽しく過ごしていました。しかし、この村には一つだけ秘密がありました。それは夜空を見上げると、一本の大きな木が空に浮かんでいるというものです。 この木は星々の間に存在し、村の住民たちはそれを「天の木」と呼びました。天の木は特別な力をもっていて、願い事をするとそれが叶うと言われていました。しかしそれを知っているのは村の長だけであり、彼はこの秘密を守るために他の人々には言いませんでした。 ある日、村の若い女性であるアリサは恋人のタカシと喧嘩をしてしまいました。タカシは仕事のために遠くへ旅立つことになり、二人の未来は不透明なものとなっていました。アリサは心配と悲しみでいっぱいになり、天の木に願い事をすることを思いつきました。 夜、アリサは一人で天の木を見上げ、涙を流しながらタカシとの幸せな未来を願いました。そして翌朝、彼女は自分が天の木の下で眠ってしまったことに気づきました。驚いた彼女は自分の周りを見渡すと、そこには見慣れない景色が広がっていました。それはタカシが働く町だったのです。 アリサはすぐにタカシを見つけ、二人は再会を喜びました。その後、二人は一緒に村に戻り、幸せな人生を送ることとなりました。そして、アリサは天の木の秘密を村の長に告げ、それ以来、村の人々は願い事が叶うという天の木の力を共有するようになりました。 これが、星に願いをという物語です。あなたが望むなら、さらに続きを作ることもできます。
```
